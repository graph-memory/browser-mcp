Understanding Accessibility Trees

Accessibility trees are how screen readers see a page. They contain a structured view of the DOM with semantic roles like button, link, textbox, and heading.

When automating a browser for an AI agent, the accessibility tree is often a better anchor than CSS selectors because it is stable across design changes.

This post walks through the basics of the tree and how to query it efficiently from a test runner.
